CSS 初始化 -- 清除浏览器设置的默认样式

@charset "UTF-8";
/* 内外边距清零 */
html, body, ul, li, ol, dl, dd, dt, p, h1, h2, h3, h4, h5, h6, form, fieldset, legend, img, input, button {
    margin: 0;
    padding: 0;
}

/* 去掉表单边框和轮廓样式*/
fieldset, img, input, button {
    border: none;
    outline-style: none;
}

/* 去掉原始列表样式*/
ul, ol {
    list-style: none;
}

input {
    font-family: "SimSun";
}

select, input {
    vertical-align: middle;
}

select, input, textarea {
    font-size: 12px;
    margin: 0;
}

/* 防止文本域拖动*/
textarea {
    resize: none;
}

/*  去掉图片底侧默认的3像素空白缝隙*/
img {
    border: 0;
    vertical-align: middle;
}

/*  表格边框合并*/
table {
    border-collapse: collapse;
}

body {
    font: 12px/150% Arial, Verdana, "\5b8b\4f53"; /*宋体*/
    color: #666;
    background: #fff;
}

/* 关闭浮动*/
.clearfix:before, .clearfix:after {
    content: "";
    display: table;
}

.clearfix:after {
    clear: both;
}

.clearfix {
    *zoom: 1; /*IE/7/6*/
}

a {
    color: #666;
    text-decoration: none;
}

a:hover {
    color: #C81623;
}

h1, h2, h3, h4, h5, h6 {
    text-decoration: none;
    font-weight: normal;
    font-size: 100%;
}

s, i, em {
    font-style: normal;
    text-decoration: none;
}

.col-red {
    color: #C81623 !important;
}

/*公共类*/
.w { /*版心 提取 */
    width: 1209px;
    margin: 0 auto;
}

.fl {
    float: left
}

.fr {
    float: right
}

.al {
    text-align: left
}

.ac {
    text-align: center
}

.ar {
    text-align: right
}

.hide {
    display: none
}



CSS 引入页面

<link rel="stylesheet" href="css/base.css">
<link rel="stylesheet" href="css/index.css">
base.css 与 index.css 不能颠倒顺序，因为加载顺序是从上往下进行加载的，后面相同的样式会覆盖掉前面相同的样式。
